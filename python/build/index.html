Instructions and Rules

Welcome to Quantum Infection

To play, find a friend! There are two players: Alice and Bob, competing to control quantum states. Alice aims to measure qubits in state 1|1\rangle1, while Bob aims for 0|0\rangle0. The game starts with a special pattern of plus (+|+\rangle+) and minus (|-\rangle) states, with corners set to 0|0\rangle0 and 1|1\rangle1. After 20 rounds, all qubits will be measured - try to control as many as possible!

Playing the Game

Each turn, you can apply one quantum gate to change the state of qubits. Single-qubit gates (H, X, Z, S) can only be applied to qubits adjacent to ones you control. Two-qubit gates (CNOT) can be used between any adjacent qubits if at least one is next to a controlled qubit. Some gates have limited uses - use them wisely! Once you start placing a multi-qubit gate, you must complete it.

Board Information

The board shows probabilities of measuring each qubit as 0|0\rangle0 (blue, Bob) or 1|1\rangle1 (red, Alice). Hover over squares to see applied gates and exact probabilities. Darker colors mean higher probability of that player winning the square. The game uses quantum superposition, so outcomes aren't certain until measurement!
